# coding-style

A central hub for managing and applying programming language-specific coding styles.

## How to use

When this skill is activated, you MUST determine the file extension(s) you are working with and read the corresponding reference file from the `references/` directory.

### Supported Languages
- **C# (.cs)**: Read `references/csharp.md`

## Guidelines
- Always prioritize the rules found in the language-specific reference file.
- If a language is not yet supported in `references/`, follow general clean code principles and existing patterns in the codebase.
