# 20250602_PXPay點加金_Ver3_Payment 服務充當金流商角色，計算此次退貨的 `(金錢, 點數)`

一、原則上，在退可被點數折抵之商品時，先退金再退點，此並非硬性規定，如果沒這麼做，外部單位並不會特別檢查。  
PM → 外部單位沒人會幫我們檢查，只能說這退款先退錢再退點，是總部財務部給的點＋金退貨規則。

二、如果此次退貨只退點數且仍有剩餘金額未退，在 InvoiceUnifer 服務仍要產生折讓檔。  

三、InvoiceUnifer 在開發票時，原本會在子單計算各自分攤的營業稅，若繼續維持此作法，會無法達成先退金再退點的原則。請 PM 問過一輪外部單位和 B2E 皆無使用，所以子單計算各自分攤的營業稅將不計算改填 0  

四、Order 服務移除在付款成功後，將點收折抵金額攤至子單的計算，又因為 B2E 有留後門可以此商品是否可被點數折抵，所以在建立訂單後，就需要記錄此子單是否可被點數折抵  

五、Order 服務傳給 Payment 服務此次要退多少錢，Payment 服務計算可退的金錢和點數後，再傳給 PXPay 和 Order 服務
